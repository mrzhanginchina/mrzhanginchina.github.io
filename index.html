Hash算法和 Hash-Map

Posted on 2018-10-10

Hash算法与Hash-Map:

什么是hash算法:

  • hash算法是将一系列的较长的二进制数据转换为较短的二进制数据的一种方法.
  • hash算法的两个特点: 不可逆和无冲突.
  • 不可逆: 当知道一个数据X的hash值之后, 不能根据hash数值反向推测出X的数值.
  • 无冲突: 如果两个数据X和Y的值不同, 则hash(X)和hash(Y)一定不相同.
  • 上面的两个特点在数学上都是不成立的, 目前通用的hash加密方法例如MD5和SHA等方法都是将逆向破解的代价变成无穷大.
  • hash算法的应用主要是校验和加密. 例如: 可以利用hash算法判断几个文件是否是同一个文件. 网站注册用户名和密码时候, 可以只保存用户密码的hash值, 通过这种方式可以有效地避免账号被盗用等不安全因素.

Hash-Map数据结构:

  • hash-map算法中, 空间一定是大于原始数据的空间大小的. 这点和hash算法正好相反, 因为hash算法是将大的数据空间映射到小的空间范围当中去.
  • 通过hash算法直接定位key值在数据库中的位置的数据结构叫做hash-map, hash-map的遍历代价可以看做是O(1), 是一种典型的空间换时间的数据结构.
  • 负载因子: 如果需要存储70个数据, 但是在内存中申请了100个空间, 那么我们可以说空间的负载因子是0.7.
  • hash-map中经常出现冲突现象, 即几个不同的key值一直映射到了相同的hash值上, 解决hash冲突主要有两种方法, 一种是hash链表, 即将固定空间变为一个链表, 如果hash值冲突, 在将新的数据附加在链表的尾部. 另外一种是hash函数双散列方法: 即在进行hash运算时候考虑碰撞次数, 通过这种方式一直找到不碰撞的存储空间.

经常用到的hash函数方法:

  • 直接寻址法：取key或key的某个线性函数值为散列地址。即H(key)=key或H(key) = a•key + b，当中a和b为常数(这样的散列函数叫做自身函数).
  • 数字分析法：分析一组数据，因此数字分析法就是找出数字的规律，尽可能利用这些数据来构造冲突几率较低的散列地址. 例如如果按照员工的的国家地来构造hash, 那么冲突的可能性非常大, 但是如果使用员工的出生的乡镇名字来构造hash, 那么冲突的可能性就大大的减小了.
  • 平方取中法：取key平方后的中间几位作为散列地址。
  • 除留余数法：取key被某个不大于散列表表长m的数p除后所得的余数为散列地址。即 H(key) = key MOD p, p<=m。不仅能够对keyword直接取模，也可在折叠、平方取中等运算之后取模。对p的选择非常重要，一般取素数或m，若p选的不好，非常容易产生同义词.

NIMA: Neural Image Assessment

总览:

这篇文章主要讲了谷歌的团队使用Inception V2的卷积设计和mobilenet的卷积设计和VGG19的方式设计了一个可以自动评估图像美观和质量的神经网络模型.
其主要的创新点是:

  1. 用输出评分分布区间的方式取代直接回归图片质量的评分均值, 因为一张图片, 不同的看会有不同的感受, 这样做更符合真实. 最后发现使用这种方式不仅可以更好地预测图像美学和质量的评分, 而且还能够更好地预测一张图片不同评分的标准差, 效果十分良好.
  2. 重新设计了一个神经网络的代价函数, 可以更加容易的进行梯度下降.
  3. 这一模型可以用于图片的自动优化, 寻找更好地图片的去噪, 饱和度和亮度等调节算法中的参数.
  4. 最后在文章末尾, 作者表示可以将整个模型当做是一个整体的代价函数, 去自动化的对图片进行美化操作.

文章的主要内容:

传统的图片质量评估的方式和优缺点.

传统的欧几里何距离衡量的图片质量有较大的不确定性, 不能准确表征图片的质量.

图像质量评估中常用的三种数据库:

A Large-Scale Database for Aesthetic Visual Analysis (AVA)

  • 这个数据库中包含255000张图片, 200位业余的摄影师对其中的图像进行打分, 并将图像排序.
  • 每张图片的分数分布从1到10分, 10分为最高质量. 全体图像的评分均值为5.5分.
  • 数据集中, 拥有极端高分和低分的图片的评分标准差较大, 对于极端图像, 人的认知也较为不同.

Tampere Image Database 2013 (TID2013)

  • 这个数据库是为了那些全参考(full-reference, 图像评估算法的一种, 拥有原始未扭曲的图像, 一般用于图像压缩算法的评估等场景)的图像评估算法设计的.
  • 数据库中包含了3000张图片, 来自于25个柯达照片. 每种图片有24种扭曲方式, 每种方式有5个等级.
  • 评分方法: 首先将五个不同压缩等级的图片中的两个进行随机的匹配, 每一对中较好的图片得一分, 较差的图片得零分.
    这样的随机成对共有九次, 所以每一个图片的分数在0到9之间.

LIVE In the Wild Image Quality Challenge Database

  • 包含由移动设备拍摄的1162张图片, 每张照片通过175个维度进行评分, 最终取平均.
  • 照片的平均分在1到10之间. 数据库仅仅提供每张照片评分的均值和标准差, 不提供原始的分数分布.
  • 在实验室, 采用最大熵的方式, 通过均值和标准差估计了每张图片的评分的具体分布.

作者提出的方法:

  • 重用VGG, mobilenet, InceptionV2在imagenet上训练的权重, 将最后一个全连接层取出, 加上一个随机初始化的, 包含十个神经元的全连接层, 通过softmax函数进行激活.
  • 训练细节: 每张图片被resize到256 256, 然后进行随机的裁剪到224 224, 同时所有的训练图像在水平方向进行翻转.
  • 作者将80% 数据设置为训练数据, 将剩下的百分之20的数据设置为测试集.
  • 代价函数
  • N: 分数的种类, 这里取10.
  • EMD: 最小的损失, 将估计的概率分布改变为原来ground truth中的概率分布.
  • CDF: 累积分布函数, , 即: 将该评分之前的所有评分加和. P_s 是属于每一个分数的概率.
  • 将r取值为2, 为了更容易进行梯度下降算法.
  • 学习率调节

  • 每经过10个epoch, 学习率衰减为0.9.

图片质量评估初识

Posted on 2018-10-08

图像质量评估(https://zhuanlan.zhihu.com/p/24804170)

图像纹理:

  • 纹理是一种反映图像中同质现象的视觉特征，它体现了物体表面的具有缓慢变化或者周期性变化的表面结构组织排列属性.
  • 纹理具有三大标志：某种局部序列性不断重复、非随机排列、纹理区域内大致为均匀的统一体.
  • 纹理不同于灰度、颜色等图像特征，它通过像素及其周围空间邻域的灰度分布来表现，即：局部纹理信息。局部纹理信息不同程度的重复性，即全局纹理信息。

孪生神经网络: Siamese network

  • 孪生神经网络是一个双路网络, 两个网络直接通过共享权值的方式来连接
  • 输入不再是单个样本，而是一对样本，不再给单个的样本确切的标签，而且给定一对样本是否来自同一个类的标签，是就是0，不是就是1
  • 设计了两个一模一样的网络，网络共享权值W，对输出进行了距离度量，可以说l1、l2等.
  • 针对输入的样本对是否来自同一个类别设计了损失函数，损失函数形式有点类似交叉熵损失.

    优点

    这个网络主要的优点是淡化了标签，使得网络具有很好的扩展性，可以对那些没有训练过的类别进行分类. 而且这个算法对一些小数据量的数据集也适用, 变相的增加了整个数据集的大小, 使得数据量相对较小的数据集也能用深度网络训练出不错的效果.

分类:

  • 图像质量评估主要可以分为三种, 全参考(Full Reference-IQA, FR-IQA)、半参考(Reduced Reference-IQA, RR-IQA)和无参考(No Reference-IQA, NR-IQA)
  • 其中, 全参考是最简单的, 意味着知道原始图像和扭曲后的图像, 模型设计等较为简单. 半参考是了解原始图像中的一些相关信息, 或者是了解图像中的一部分.
  • 无参考图像评估是最有难度的一种评估方式, 也是使用最多的评估方式, 意味着只知道扭曲后的图像, 只对单一图像进行评价.

如何衡量一个评价的优劣:

  • 通常比较模型客观值与观测的主观值之间的差异和相关性
  • 常见的2种评估指标是线性相关系数（Linear Correlation Coefficient, LCC）和Spearman秩相关系数（Spearman’s Rank Order Correlation Coefficient, SROCC）
  • 第一种比较了模型预测值和真实值之间的相关性, 使用pearson系数的方式.
  • 第二种衡量了算法预测的单调性, 使用的是预测值在整体数据库中预测值的排序.

  • 新的方法: 首先把RGB图像转换到YIQ颜色空间，从中提取3000个特征，然后使用L-BFGS算法训练一个稀疏的自动编码器，输入数据是s×3000的矩阵，s表示训练样本的个数，设计一个3层的DNN，使用刚才训练的自动编码器初始化DNN，然后使用线性函数计算输出，最后根据损失函数使用反向传播算法微调DNN每层的权重。实验结果表明DIQI的有效性，并且对比经典的FR-IQA、RR-IQA算法，DNN是IQA研究中一个有前景的方向。
